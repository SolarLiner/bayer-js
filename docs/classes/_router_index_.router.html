addRoute

  • addRoute(path: string, ...methods: HTTPVerb[]): object
    • Defined in router/index.ts:209

    Add a route to manage through the router.

    The actual callback will be provided through using the returned function use, which itself returns this very instance; this way you can chain several route definitions in one go.

    Parameters

    • path: string

      Route path. Can have named parameters, which will be matched using path-to-regexp, just like many other server libraries.

    • Rest ...methods: HTTPVerb[]

      Methods to match the route to. If omitted, all methods will be matched to this route.

    Returns object
